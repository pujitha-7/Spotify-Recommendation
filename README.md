# Spotify-Recommendation
🎵 Spotify Recommendation System using Machine Learning. This project groups songs using KMeans clustering and recommends similar songs based on audio features.
Dataset:
This project uses a CSV dataset containing song features like danceability, energy, tempo, etc.
🎵 Spotify Recommendation System

This project uses Machine Learning (KMeans Clustering) to recommend similar songs.

📊 Dataset:
Contains song features like danceability, energy, loudness, tempo, etc.

⚙️ Technologies Used:
- Python
- Pandas
- Scikit-learn

🚀 Output:
User enters a song ID and gets similar song recommendations.
